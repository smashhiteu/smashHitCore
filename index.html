Ontology Specification Draft

Abstract

The smashHitCore ontology extends the scope of existing ontologies by providing an integrated semantic model for multiple domains such as consent, contracts, licenses and sensor data. This work is part of the smashhit project, which is developing a framework for trusted and secure sharing of data streams from both personal and industrial platforms needed to build sectorial and cross-sectorial services. The ontology models multiple GDPR legal bases such as consent and contracts, but is not limited to them. Concepts such as metadata, sensor data, data processing, involved entities, which can be useful for compliance checking, are modelled as well.

: Description back to ToC

smashHitCore consists of 1135 axioms, 134 classes, 32 object properties, 23 data properties and reuses 9 ontologies: GConsent, DPV, FIBO, PROV-O OntoSensor, schema.org, Data Catalog Vocabulary (DCAT), CampaNeo , and Languages, Countries, and Codes (LCC).
